================================================================================
  八、给组员的说明
================================================================================

  1. 所有 .txt 文档使用 UTF-8 编码，可以用记事本/VS Code 打开
     如遇乱码: 用 VS Code → 右下角点击编码 → 选择 "通过编码重新打开" → UTF-8

  2. 视频文件建议用 VLC / PotPlayer 打开，支持逐帧查看

  3. 代码不能直接运行的原因:
     - 模型权重未包含在压缩包中（太大）
     - pickle 文件未包含（太大）
     - 但代码逻辑完整，文档中有对应说明

  4. 最终报告和 PPT 的文字内容:
     01~08 系列文档提供了完整的中文说明，
     可以直接从中提取段落用于 Word 报告和 PPT 文案

  5. 关于 Stage 3:
     如需实际运行 Stage 3 生成报告，需要:
     a. 安装 Conda + 配置 ai_lab 环境
     b. 安装 Ollama + 拉取 qwen2.5:7b
     c. 有 Stage 2 输出的 st_graph_output.json
     详见 07_运行指南.txt
