================================================================================
  六、Stage 2 — Phase 4: M2 曲率积分转弯判定
================================================================================

  ┌──────────┬──────────┬──────────┬──────────┬──────────┬──────────┬──────────┐
  │ 视频     │ ε (度)   │ 评估轨   │ 直行     │ 左转     │ 右转     │ 转弯比   │
  │          │ 自适应   │ 迹数     │          │          │          │          │
  ├──────────┼──────────┼──────────┼──────────┼──────────┼──────────┼──────────┤
  │ S05_c028 │ 85.94°   │ 184      │ 149(81%) │ 23(12%)  │ 12(7%)   │ 19.0%    │
  │          │ (钳位上限)│          │          │          │          │          │
  ├──────────┼──────────┼──────────┼──────────┼──────────┼──────────┼──────────┤
  │ S06_c041 │ 35.56°   │ 231      │ 197(85%) │ 18(8%)   │ 16(7%)   │ 14.7%    │
  ├──────────┼──────────┼──────────┼──────────┼──────────┼──────────┼──────────┤
  │ S06_c043 │ 28.43°   │ 214      │ 180(84%) │ 14(7%)   │ 20(9%)   │ 15.9%    │
  ├──────────┼──────────┼──────────┼──────────┼──────────┼──────────┼──────────┤
  │ S06_c046 │ 65.78°   │ 132      │ 116(88%) │ 10(8%)   │ 6(5%)    │ 12.1%    │
  └──────────┴──────────┴──────────┴──────────┴──────────┴──────────┴──────────┘

  ε 计算: W = Σ ω_t (逐帧转角积分), ε = P85(|W|), clamped [0.12, 1.5] rad
  判定: W > +ε → Right Turn, W < -ε → Left Turn, else → Go Straight
  S05_c028 ε 被钳位至上限 (85.94°) — 该场景转弯幅度极大，阈值触顶
